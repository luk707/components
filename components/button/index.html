Custom component

You may wish to use your own component but have it styled as a button. To do so use the buttonStyled funtion and pass it your own component:

1
2
3
4
5
6
import { buttonStyled } from '@luk707/components/lib/button'

const DivButton = buttonStyled('div')

export default () =>
  <DivButton prominence="primary">Hello world</DivButton>
